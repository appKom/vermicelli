# Algoritme

Algoritmen baserer seg på MIP-programmering (Mixed Integer Linear Programming).

## Setup Python Venv

```bash
cd algorithm
python -m venv ".venv"
```

```
.\.venv\Scripts\activate
pip install -e .
pip install -r requirements.txt
```
